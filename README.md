# Shreya Portfolio

## Requirements
- Node.js 20 or newer
- npm

## Run locally

```bash
npm install
npm run dev
```

Open http://localhost:3000

## Production

```bash
npm run build
npm start
```

## Deploy to Vercel

1. Push this project to GitHub.
2. Import the repository into Vercel.
3. Framework: Next.js (auto detected).
4. Click Deploy.

No additional configuration is required.
